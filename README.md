# React Portfolio

A clean and modern personal portfolio built with React. This project highlights skills, projects, and contact details in a polished single-page layout.

## Live Demo
https://vishwas-portfolio-1.netlify.app/

## What’s included
- Responsive homepage with animated sections
- Skills showcase with progress visuals
- Project section with links and details
- Contact form powered by EmailJS
- Dark-style theme with refined card backgrounds

## Tech stack
- React
- React Bootstrap
- EmailJS for contact form delivery
- CSS custom properties for theme styling

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Add environment variables in a `.env` file:
   ```env
   REACT_APP_EMAILJS_SERVICE_ID=your_service_id
   REACT_APP_EMAILJS_TEMPLATE_ID=your_template_id
   REACT_APP_EMAILJS_PUBLIC_KEY=your_public_key
   ```
3. Start locally:
   ```bash
   npm start
   ```
